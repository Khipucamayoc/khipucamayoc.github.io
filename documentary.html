Quipu Project

Using Python to create an interface to make simplified graphics of these pre-Columbian devices.

I programmed the "Khipu Visualizer" from scratch, and it offers an intuitive user interface that allows archaeologists, artists, or whoever is interested in doing a simple graphic of a quipu based on specific data, being this observational, or acquired from sites such as "The Quipu Database".

This program was conceptualized by Paola Torres for her "Knots of Code" project.
